创建软件快捷方式

目前用过的三种快捷启动的方式：创建软连接到/usr/local/bin、在shell配置文件中添加alias、在/usr/share/applications中创建.desktop的启动器

  • 软连接
cd /usr/local/bin
sudo ln -s /opt/idea-IU/bin/idea.sh idea
sudo chmod +x idea
  • ~/.zshrc中添加alias (本人使用的zsh)
alias idea="/opt/idea-IU/bin/idea.sh"
  • 创建.desktop启动器
cd /usr/share/applications
sudo touch idea.desktop
sudo vim idea.desktop

然后再idea.desktop中添加如下配置

[Desktop Entry]
Name=idea
Exec=/opt/idea-IU/bin/idea.sh
Type=Application
StartupNotify=false

通过软连接的方式在$Path下创建软连接，可以在各种终端中直接使用

通过在/.zshrc中添加别名只在zsh终端中生效（不同种终端对应各自的/.*shrc）

创建.desktop启动器可以通过菜单中找到idea启动应用（eg. roki -show drun）

设置屏幕亮度

# 先通过指令查看输出信号的名字 然后设置显示器的亮度0-1.0
xrandr --listmonitors

xrandr --output eDP-1 --brightness [0 - 1]
