SpringBoot读取yml配置的几种方式

2024/3/6 SpringBoot

三种方式@Value、@ConfigurationProperties、@Environment

@Value

在属性上添加@Value注解（需要属性所在类是一个bean）

@Value(("${upload.aliyun.bucket}"))
private String bucket;

@ConfigurationProperties

如果需要一个bean来专门配置映射的话，可以使用@ConfigurationProperties来读取 (因为是JavaBean，所以被@Bean注解的方法也是可以添加@ConfigurationProperties注解来赋值的)

@Data
@Configuration
@ConfigurationProperties(prefix = "upload.aliyun")
public class AliyunConfigProperties {
    private String accessKeyId;

    private String secretKey;

    private String endpoint;

    private String region;

    private String bucket;
}

然后在使用到这个bean的类中添加@EnableConfigurationProperties(AliyunConfigProperties.class)来读取bean中的值。

@Configuration
@EnableConfigurationProperties(AliyunConfigProperties.class)
public class AliyunConfig {
    //
}

@Environment

@Autowired
private Environment env;

@RequestMapping(value = "index2", method = RequestMethod.GET)
public void index2() {
    System.out.println(env.getProperty("test.msg"));
}
